这是一个用python编写的神经网络程序，用以从图片中识别人脸。

使用了OpenCV，以及numpy，PIL包，和python的标准库。

参考了bytefish大神提供的教程，大量引用了facerecognition_guide中的代码。

这个程序原本是用来交课堂作业的，但是本着开源精神就发上来了。大家有爱自取。
